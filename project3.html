Employee Management System

This interactive Python demo lets you explore the concepts of object-oriented programming (OOP).
It features an`Employee` class and a subclasses 'Manager'.
You can test inheritance, encapsulation, getters, setters, and methods directly in your browser.
What this program can do:

  • Set employee's name, salary and position
  • Method to give a raise based on the performanceand years worked.
  • Subclass"Manager" that can manages a team of employees and assign tasks.

Modify the code in the editor, click "Run Code," and observe how the output changes. Hope that you will like the program!
