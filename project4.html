School System Simulation

This interactive Python demo lets you explore the concepts of object-oriented programming (OOP).
It features a `Person` class and two subclasses:
`Teacher`and `Student`. You can test inheritance, encapsulation, getters, setters, and methods directly in your browser.
What this program can do:

  • Set person's name and age
  • Set grades for students and subjects for teachers.
  • Method to allow students to enroll in classes.
  • Method to let teachers assign grades for students.

Modify the code in the editor, click "Run Code," and observe how the output changes. Hope that you will like the program!
