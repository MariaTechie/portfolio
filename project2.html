Library Management System

This interactive Python demo lets you explore the concepts of object-oriented programming (OOP).
It features a `Book` class and 'Library' class. You can test inheritance, encapsulation, getters, setters, and methods directly in your browser.
What this program can do:

  • Set book's title author's name and the status
  • Method to add a novel to the Library

Kindly note that this program is in progress and not complete, hopefully more methods will be added to make it more dynamic very soon.

Modify the code in the editor, click "Run Code," and observe how the output changes. Hope that you will like the program!
